# v0.3.0

Release date: 2026-07-11

## New features

- **逻辑时间冻结**：暂停游戏逻辑更新，输入仍保持响应。仅限 RA3 1.12 profile。
- **慢动作（50% 速度）**：以半速运行游戏逻辑。仅限 RA3 1.12 profile。
- **选中单位统一执行框架**：统一选中单位修改的执行入口，提升一致性与可维护性。
- **选中单位独立选项卡**：'选中单位·生命值/速度/其他' 和 '伤害与无敌' 移至独立侧边栏选项卡，主页从 8 组 39 项精简为 4 组 21 项。

## Improvements

- 协议升级至 v9。
- 代号从 Ra3Trainer 更名为 RayaTrainer。
- RA3 1.12 Hook 解析切换为签名优先策略。
- 资产包系统上线：日冕数据从磁盘加载。
- 架构优化：抽取诊断状态、去重 Hook 计数、Agent 工程加入主 solution。
- 跨语言契约一致性检查工具加入 CI 流程。
- 选中单位功能验证覆盖通过：测试全部通过。

## Limitations

- 逻辑时间冻结与慢动作仅在 RA3 1.12 通过实机验证。其他 profile（1.13、Uprising 1.0、Uprising 1.1）尚缺等价 Hook 签名。
- 攻击范围扩展与攻击速度调整为隐藏功能，不列入公开 UI 或受支持的功能范围。这些功能的实现架构已完成验证，但未完成 UX/产品化打磨，不在首次公开发布范围。
- 本工具不支持联机/多人模式、DRM 绕过、反作弊绕过或任何形式的访问限制规避。
